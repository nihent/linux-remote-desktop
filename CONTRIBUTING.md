Contributing

Thank you for your interest in contributing to Linux Remote Desktop!

How to Contribute

1. Fork this repository.
2. Create a new branch.
3. Make your changes.
4. Commit your changes.
5. Push to your fork.
6. Open a Pull Request.

Reporting Bugs

If you find a bug, please open an Issue with:

- Operating System
- Steps to reproduce
- Expected behavior
- Screenshots (if applicable)

Feature Requests

Feature requests are always welcome. Open an Issue and describe your idea.

Thank you for helping improve this project! ❤️
